# 概率与随机

> 综合多本书的跨书主题文章。

## 定义

**概率与随机**是游戏数值与模拟的基础工具：从掉落概率、伤害随机，到自然感的运动（噪声驱动的流场）、群体行为。关键区分是**均匀随机**（每个值等概率，显得机械）与**有结构的随机**（正态分布、Perlin 噪声，显得自然）。

## 多本书视角对比

- **《代码本色》**（[参考](../references/代码本色.md)）：Shiffman 把随机性作为全书第 0 章地基。系统讲解：均匀分布、非均匀分布、概率事件、正态（高斯）分布、自定义分布随机数，以及 **Perlin 噪声**——能产生自然连续平滑随机的算法，是流场、纹理、地形生成的核心。强调"自然感"模拟需要结构化随机而非纯随机。
- **《游戏中的数学与物理学》**（[参考](../references/游戏中的数学与物理学（第2版）.md)）：从数学视角讲概率统计基础，并用于掉落、伤害、A* 启发式等游戏算法。

## 方法/模型

- **均匀随机**：每个值等概率，用于简单掉落、洗牌。
- **正态分布（高斯）**：钟形分布，生成聚集于均值的自然随机（如身高、误差）。
- **自定义分布**：拒绝采样等方法生成任意分布的随机数。
- **Perlin 噪声**：平滑连续伪随机，用于流场、地形、纹理；关键参数是噪声频率与映射。
- **概率事件**：`if (random() < p)` 判定事件是否发生，用于掉落、暴击。

## 具体公式与例子

**正态分布采样**（Box-Muller 变换）：从两个均匀随机数 `u1, u2 ∈ (0,1]` 生成标准正态：

```
z0 = sqrt(-2·ln(u1)) · cos(2π·u2)
z1 = sqrt(-2·ln(u1)) · sin(2π·u2)
```

`z0, z1` 服从 N(0,1)，乘 σ 加 μ 即得任意正态分布——用于生成"误差""弹道散布"等自然随机。

**拒绝采样做非均匀分布**（代码本色的图示）：要按某条曲线 `y=f(x)` 分布采样，先在包围矩形内均匀投点，只保留 `random() < f(x)` 的点。这能生成任意形状的分布——例如靠近中心更密的随机走位。

**Perlin 噪声 vs 均匀随机**对比：

| | 均匀随机 random() | Perlin 噪声 noise(t) |
|---|---|---|
| 相邻值关系 | 无关，剧烈跳变 | 平滑连续，自相似 |
| 视觉 | 雪花噪点 | 自然纹理/地形 |
| 用途 | 洗牌、掉落 | 流场、云、海岸线 |

**1D 噪声驱动运动**：`x += (noise(t) - 0.5) * scale`，让物体随机游走但运动平滑——比 `x += random(-1,1)` 自然得多，是"自然感"的核心技巧。

**多倍频（fractal）噪声**：叠加多个不同频率/振幅的 Perlin 噪声 `Σ noise(fᵢ·t)/2ⁱ`，生成既有大尺度结构又有细节的程序化地形。

## 应用

- 掉落、暴击、闪避用概率事件 + 权重（见批次②数值公式体系）。
- 自然运动（烟雾飘散、流场、地形）用 Perlin 噪声。
- 群体行为（鸟群抖动、生物游动）用噪声驱动方向。
- 程序生成（地图、纹理）用噪声 + 多倍频。
