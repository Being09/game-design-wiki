# 群体行为

> 综合多本书的跨书主题文章（主要来源：《代码本色》）。

## 定义

**群体行为**是用局部简单规则涌现出整体复杂群体运动（鸟群、鱼群、人群）的模拟技术。源自 Craig Reynolds 的 **Boids** 模型与**转向行为（Steering Behaviors）**理论：每个个体（自治智能体）只感知局部邻居，按几条简单规则产生转向力，整体便涌现出有机的群聚、分流、避障行为。

## 多本书视角对比

- **《代码本色》**（[参考](../references/代码本色.md)）：第 6 章系统讲解。先把物体升级为自治智能体（Vehicle 类），核心转向力公式 `转向力 = 所需速度 − 当前速度`，受 maxspeed/maxforce 约束。覆盖 seek/arrive/flee/pursue/flow field/path following，最后用**分离、对齐、聚集**三规则组合出 Boids 群集。强调"无中心领导者、局部规则涌现整体"。
- 本概念在批次③另一书（《数学与物理学》）中无对应专论（该书偏刚体/碰撞物理），故概念主要来自代码本色。

## 方法/模型

- **转向力公式**：`转向力 = 所需速度 − 当前速度`，受最大速率与最大转向力限制。
- **自治智能体**：有有限感知、能自主决策产生力、无中心领导者。
- **Boids 三规则**：分离（separation，避开拥挤邻居）、对齐（alignment，匹配邻居平均方向）、聚集（cohesion，朝邻居中心移动）。
- **流场（Flow Field）**：智能体跟随 Perlin 噪声向量场，产生自然的群体流动。
- **路径跟随**：智能体沿多段路径运动，可预测未来位置做平滑跟随。
- **性能优化**：用距离平方代替开方、三角函数查表、避免不必要的对象创建。

## 具体例子

**转向力公式的实现**（seek 行为——朝目标冲）：

```
desired = target - position      # 指向目标的向量
desired.normalize()
desired *= maxspeed              # 期望速度 = 方向 × 最大速率
steer = desired - velocity       # 转向力 = 期望 - 当前
steer.limit(maxforce)            # 限制转向力（否则瞬间转向不自然）
velocity += steer
velocity.limit(maxspeed)
position += velocity
```

关键在 `steer.limit(maxforce)`——没有它物体会瞬间转向目标，像磁铁吸过去；有限转向力让运动有惯性、像生物。

**Boids 三规则的向量直觉**：

```
分离：sum += (self.pos - neighbor.pos).normalize() / dist²
        # 离得越近的邻居排斥力越大（除以距离平方）
对齐：avg_vel = mean(neighbor.velocities)
        steer = avg_vel - self.vel
聚集：center = mean(neighbor.positions)
        steer = seek(center) - self.vel
```

三个转向力加权求和（分离通常权重最高，否则挤成一团），每只鸟只看半径内的邻居，整体涌现出有机群飞——没有一只鸟知道"整个群体"长什么样。

**性能优化（为什么 Boids 是 O(n²) 瓶颈）**：n 只鸟两两判定邻居需 n²/2 次。100 只还行，1000 只就 50 万次/帧。解法：
- **空间划分**：四叉树/网格，只判同网格邻居，降到 O(n·log n)。
- **距离平方**：`if (d² < radius²)` 替代 `if (d < radius)`，省掉每个邻居一次 sqrt。
- **视野锥剔除**：先点积判方向再算距离，跳过身后的鸟。

**涌现的魔力**：只调三规则权重就能产生完全不同的群体——分离权重高 → 松散鱼群；对齐权重高 → 整齐雁阵；聚集权重高 → 密集蜂群。无需任何"群体智能"代码，局部规则自己涌现。

## 应用

- 鸟群、鱼群、兽群等自然群体用 Boids。
- RTS 单位寻路后的自然运动用转向力 + 避障。
- 人群疏散、交通流模拟。
- 流场驱动的粒子或生物游动（配合 Perlin 噪声）。
