# 机制设计

> 综合多本书的跨书主题文章。

## 定义

**机制设计**是博弈论的"逆向工程"——给定希望达到的目标（如社会福利最大化、收益最大化），设计博弈规则使策略型参与者在自利行为下达到该目标。核心工具是**显示原理**（任何均衡都可由说真话的直接机制复制）与**激励相容（IC）**约束。又称"逆向博弈论"。

## 多本书视角对比

- **《博弈论》**（[参考](../references/博弈论.md)）：第 7 章系统讲解。显示原理把机制设计归约为直接显示机制；研究最优机制（IR/IC 约束、Myerson 最优拍卖）、Myerson–Satterthwaite 效率不可能定理、收益等价定理、拍卖收益最大化。
- **《算法博弈论二十讲》**（[参考](../references/斯坦福算法博弈论二十讲.md)）：第 2-10 章，从计算机科学视角，更注重计算高效性。迈尔森引理（分配规则可实施 ⇔ 单调）、VCG 机制、近似最优拍卖（预知不等式、Bulow-Klemperer）、频谱拍卖、无货币机制（肾脏交换、稳定匹配）。

两书互补：经典教材给数学严谨与最优性定理，算法博弈论给计算可行性与实际应用。

## 方法/模型

- **显示原理**：任何机制均衡都可由说真话的直接显示机制复制。
- **激励相容（IC）**：机制使如实报告类型成为最优选择；DSIC（占优策略 IC）最强。
- **参与约束（IR）**：保证每类型参与不劣于保留效用。
- **VCG 机制**：选福利最大化分配，每人支付其外部性，实现 DSIC + 福利最优。
- **Myerson 最优拍卖**：用虚拟估值把收益最大化归约为虚拟福利最大化。
- **收益等价定理**：标准假设下多种拍卖给卖方相同期望收益。

## 具体例子

**VCG 外部性支付的实例**：拍卖一件物品给三人，估值 v1=100, v2=80, v3=50。
- 福利最大化分配：给 v1（最高），福利=100。
- 玩家 1 的外部性 = "没有他时其他人的最大总福利" = max(v2,v3) = 80。
- 玩家 1 支付 80（其外部性），净效用 100−80=20。

为什么这能 DSIC？若玩家 1 谎报 70（低于真实 100）：他输给报 80 的玩家 2，得 0 < 20，亏。若谎报 120：仍赢，仍付外部性 80（外部性只取决于别人，与他自己的报价无关），净效用不变。任何谎报都不能改善——说真话是占优策略。

**VCG 的计算关键**：每个人的支付 = "没有他时的社会福利 − 其他人在他参与时获得的福利"。只依赖别人的估值，不依赖自己的报价——这是 DSIC 的数学根源。

**Myerson–Satterthwaite 不可能性定理**：在双边交易（卖方有成本、买方有估值，都私有）下，**没有任何**机制能同时满足 DSIC + 个人理性 + 预算平衡 + 社会效率。必有取舍——通常牺牲预算平衡（机制要补贴）或效率。这是机制设计的根本限制，解释了为什么"完美"的交易机制不存在。

**显示原理的力量**：看起来机制可以有任意复杂的策略空间（多轮报价、动态调整）。但显示原理证明：任何机制的任何均衡，都存在一个"玩家直接报真实类型、机制按规则输出结果"的等价直接机制。所以机制设计只需研究"说真话的直接机制"，大幅缩小搜索空间。

**游戏设计中的机制设计思维**：不是字面上的拍卖，而是"设计规则使玩家自利行为导向设计者目标"。例如：
- 想让玩家组队 → 设计"组队收益 > 单人收益"（激励相容）。
- 想防止囤积 → 设计"持有成本"（仓库租金、衰减）。
- 想引导探索 → 设计"边际收益递增"（奖励新区域而非重复）。
把"我希望玩家做 X"翻译成"在什么规则下，玩家做 X 是占优策略"，就是机制设计。

## 应用

- 拍卖设计（在线广告、频谱）用 VCG / Myerson / SAA。
- 无货币分配（肾脏交换、择校）用稳定匹配 / TTC。
- 游戏内经济规则设计：使玩家自利行为导向设计者目标（见 [[虚拟经济系统]]）。
- 机制设计理论与游戏机制设计（批次① design pattern）交叉。
